# ejercicio parcial 1
Mi primer proyecto de React con Expo GO


1. Crear proyecto: **npx create-expo-app parcial1 --template**
2. Seleccionar **Blank / En blanco**
3. Escribir el nombre de tu aplicación
1. Crear una vista de navegación con Tabs o Pestañas donde los botones tengan el mismo width (usando flex) y estén acomodados en una fila al fondo de la página. Serán 3 botones, Cada uno mantendrá aproximadamente 33.3% del ancho de la pantalla, sin importar el contenido, nunca debe de verse uno más grande que otro, los botones deben de decir lo siguiente: Miki, Choy, Mitzu (es el nombre de mis 3 gatos)
2. Crear una navegación donde al hacer clic en cada tab (es decir, cada boton) muestre el nombre del componente (nombre de cada uno de mis gatos y una foto del mismo). Enviar este titulo de componente a través del parámetro en la ruta. Se debe de ver que la ruta está cambiando. Quiero que diga el nombre de la mascota en cada una de las rutas 
3. Quiero que sea en una colorimetría verde y azul calidos
4. Dame el espacio de añadir una imagen en cada pestaña, en cada una podré una imagen del gato que corresponda
